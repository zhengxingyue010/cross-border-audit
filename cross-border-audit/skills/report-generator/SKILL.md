# 报告生成 Skill

## 功能
基于风险识别结果，生成最终的风险审计报告。

## 输入
- 标记风险后的交易数据
- 风险模式匹配结果
- 新发现模式（如果有）

## 报告结构

### 1. 执行摘要
- 审计时间范围
- 总订单数
- 发现风险订单数
- 风险等级分布
- 关键发现概览

### 2. 风险详情
按风险等级分类：
- **高风险** 🔴
  - 风险描述
  - 涉及订单
  - 建议措施

- **中风险** 🟡
  - 风险描述
  - 涉及订单
  - 建议措施

- **低风险** 🟢
  - 风险描述
  - 涉及订单
  - 建议措施

### 3. 新模式发现
- 新发现的风险模式
- 发现过程说明
- 建议是否纳入模式库

### 4. 统计分析
- 风险趋势图表
- 国家分布
- 货物分布
- 时间分布

### 5. 建议与行动计划
- 短期措施（立即执行）
- 中期措施（1-2周）
- 长期措施（1个月+）
- 模式库更新建议

## 输出
- 格式：Markdown
- 保存位置：`reports/` 目录
- 文件名格式：`审计报告_YYYY-MM-DD_HHMMSS.md`

## 风险评分标准
- 0-30 分：低风险
- 31-60 分：中风险
- 61-80 分：高风险
- 81-100 分：严重风险
