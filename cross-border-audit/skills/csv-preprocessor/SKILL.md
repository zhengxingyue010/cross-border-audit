# CSV 预处理 Skill

## 功能
读取跨境电商交易订单 CSV，进行数据清洗和标准化。

## 输入字段
- 发票代码
- 货物代码
- 货物名称
- 数量
- 单价
- 国家
- （其他可能存在的字段）

## 处理步骤

### 1. 数据读取
- 从 `data/raw/` 读取原始 CSV
- 自动检测编码（UTF-8, GBK, GB2312）
- 处理表头和数据行

### 2. 数据清洗
- 去除空行、重复行
- 标准化字段名（统一中文/英文）
- 处理缺失值：
  - 关键字段缺失 → 标记并记录
  - 可填充字段 → 用默认值填充

### 3. 数据标准化
- 数量：转换为整数，去除单位
- 单价：转换为浮点数，统一货币
- 国家：标准化国家名称（使用 ISO 3166）
- 日期：统一格式为 YYYY-MM-DD

### 4. 数据验证
- 检查数量 > 0
- 检查单价 > 0
- 检查国家名称有效性

## 输出
- 保存到 `data/processed/` 目录
- 文件名格式：`processed_原始文件名.csv`
- 生成清洗报告：`data/processed/清洗报告_原始文件名.md`

## 使用方式
```
调用此 skill 时提供：
- CSV 文件路径（相对或绝对）
- 可选：指定需要的字段列表
```
