# 风险识别 Skill（带记忆机制）

## 功能
基于风险模式库识别交易风险，自动发现新模式并补充到模式库。

## 工作流程

### 阶段 1：加载记忆
1. 读取 `risk-patterns/patterns.json` - 已知风险模式
2. 读取 `risk-patterns/lessons-learned.md` - 历史经验教训

### 阶段 2：已知模式识别
对每条交易记录，应用所有已知模式：
- PATTERN-001: 同一货物高频发往同一国家
- PATTERN-002: 单价异常波动
- PATTERN-003: 整数数量可疑订单
- ... 及其他模式库中的模式

### 阶段 3：新模式发现（短期记忆）
分析数据特征，寻找异常模式：
1. **统计异常检测**
   - 偏离均值超过 3σ 的记录
   - 价格分布异常点
   - 数量分布异常点

2. **关联规则挖掘**
   - 货物-国家组合异常频繁
   - 发票代码关联异常
   - 时间序列异常

3. **相似度分析**
   - 与已知风险案例相似度 > 80%
   - 新的可疑模式

### 阶段 4：模式库更新（长期记忆）
如果发现新模式：
1. 生成模式 ID（PATTERN-XXX）
2. 提取模式规则和特征
3. 人工确认后写入 `risk-patterns/patterns.json`
4. 记录到 `risk-patterns/lessons-learned.md`
5. 保存发现详情到 `risk-patterns/new-discoveries/`

## 输出
- 风险标记的交易数据
- 新模式发现报告
- 更新后的风险模式库（如果有新发现）

## 记忆机制
```
每次运行：
  读入旧模式 → 识别风险 → 发现新模式 → 写入新模式
  ↓
下次运行：
  读入更新后的模式 → ... 循环往复 ...
```
